# Coolatro Mod

![logo](assets/2x/icon.png)
**A Vanilla-adjacent mod with cool jokers to experiment with!**

(Requires Steammodded)

If you want to keep vanilla experience, but want to try out new strategies this mod is for you!
Try out playing different hand combinations and try to turn downsides into powerful strenghts in your deck!

| !(assets/2x/j_cl_caleidoscope.png) | !(assets/2x/j_cl_stamped_delivery.png) | !(assets/2x/j_cl_retrograde.png) |
| If played hand is Flush\nthis scores, gains X0.1 Mult\nand changes suit of played hand\n(Currently X0 Mult) | 4, 5, 6, 7 gain a random seal\nwhen added to a deck | Played hands above level one\ndecrease poker hand and\ncreate Trance spectral card\n(Must have room) |

For now this mod gives 15 jokers (with intetnt being updated to 30 in the future).
This mod is balanced around vanilla game and Ante 8, however some of jokers can eaisly be built around for infinity games.

This mod is still experimental and work-in-progress :)
Feel free to report any bugs or balance changes proposals!
